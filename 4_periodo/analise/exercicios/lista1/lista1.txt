# Lista 1

## 1 ) Liste as possíveis classes com base no minimundo, referente a Objetos, Eventos e Agentes;

### Possíveis classes e atributos:

#### Agentes:
1. Pessoa
  - nome: string
  - telefone: string
  - email: string
  - foto: string
  - endereco: string
  - inseridoPor: Funcionario

2. Restaurante extends Pessoa

3. Funcionario extends Pessoa

4. Gerente extends Funcionario
  · reservarMesa()
  · cancelarReservaMesa()
  · cadastrarFuncionario()

5. Cozinheiro extends Funcionario
  · elaborarPedido()
  · listarPedidosPendentes()
  · finalizarElaboracaoPedido()

6. Garcom extends Funcionario
  · registrarPedido()
  · cancelarPedido()
  · listarPedidos()
  · consultarInfoPedido()
  · consultarProdutoPedido()
  · entregarPedido()
  · calcularValorTotalMesa()

7. Caixa extends Funcionario
  · realizarPagamento()

#### Objetos
1. Produto
  - nome: string
  - valor: number
  - quantidade: number
  - tempoMedioPreparo: number
  - tempoMaxPreparo: number
  - inseridoPor: Funcionario

2. TipoProduto
  - nome: string
  - inseridoPor: Funcionario

3. Mesa
  - numero: number
  - inseridoPor: Funcionario

#### Eventos
1. MesaReservada
  - dataReserva: string
  - horaReserva: string
  - nomeCliente: string
  - telefoneCliente: string
  - inseridoPor: Funcionario

2. Pedido
  - data: string
  - hora: string
  - produto: Produto
  - quantidade: number
  - valor: number
  - valorTotal: number
  - inseridoPor: Funcionario

3. Comanda
  - dataAbertura: string
  - horaAbertura: string
  - pedidos: Array<Pedido>
  - mesa: Mesa
  - valor: number
  - pagamento: Pagamento
  - dataFechamento: string
  - horaFechamento: string
  - inseridoPor: Funcionario

4. Pagamento
  - dataPagamento: string
  - horaPagamento: string
  - comissao: boolean
  - valor: number
  - tipoPagamento: ETypePagamento
  - formaPagamento: PagamentoCartao | PagamentoDinheiro
  - inseridoPor: Funcionario

5. PagamentoDinheiro
  - dataPagamento: string
  - horaPagamento: string
  - data: string
  - total: number
  - pago: number
  - troco: number
  - inseridoPor: Funcionario

6. PagamentoCartao
  - dataPagamento: string
  - horaPagamento: string
  - titular: string
  - cpfTitular: string
  - numero: string
  - dataVencimento: string
  - inseridoPor: Funcionario

I. ETypePagamento
  - CARTAO
  - DINHEIRO
